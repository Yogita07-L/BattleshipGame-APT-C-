# Battleship Game – Lab Project

An object-oriented C++ implementation of the classic Battleship game, developed as part of university coursework and later published for portfolio purposes.

This project models the core mechanics of Battleship, including ship placement, shot handling, and board visualization using clean object-oriented design principles.

---

## Technologies & Skills

- C++
- Object-Oriented Programming (OOP)
- STL (vector, map, set)
- Unit Testing
- Eclipse IDE
- Git & GitHub
- Software Design & Modeling (UML)

---

## Features

- Immutable `GridPosition` and `Ship` classes
- Rule-based ship placement validation
- Detection of hits, misses, and sunken ships
- Console-based game board display
- Opponent tracking system
- Automated unit tests for core logic
- Modular and maintainable code structure
